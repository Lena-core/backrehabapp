# Звуковые файлы для Android

Поместите сюда следующие MP3 файлы:

## prepare.mp3
Мягкий, спокойный звук для начала подготовки к упражнению.
Рекомендации: тихий колокольчик, мягкий сигнал, длительность 1-2 сек.

## start.mp3  
Энергичный, мотивирующий звук для начала выполнения упражнения.
Рекомендации: четкий звуковой сигнал, бодрый тон, длительность 1-2 сек.

## rest.mp3
Расслабляющий звук для перехода к отдыху между подходами.
Рекомендации: мягкий, успокаивающий тон, длительность 1-2 сек.

## next_set.mp3
Мотивирующий звук для начала нового подхода после отдыха.
Рекомендации: ободряющий сигнал, средней интенсивности, длительность 1-2 сек.

## complete.mp3
Победный звук завершения всего упражнения.
Рекомендации: радостный, триумфальный звук, длительность 2-3 сек.

## warning.mp3
Предупреждающий звук за 3 секунды до окончания фазы.
Рекомендации: внимательный, но не агрессивный сигнал, длительность 0.5-1 сек.

## tick.mp3
Короткий звук секундной отметки для обратного отсчета.
Рекомендации: короткий щелчок или тик, длительность 0.1-0.3 сек.

## Где найти звуки:
- freesound.org (бесплатные звуки)
- zapsplat.com 
- Adobe Stock Audio
- Можете записать собственные

## Формат:
- MP3, 44.1 kHz
- Качество: 128-320 kbps
- Длительность: указана для каждого файла
